# 实现重点特性

## 目的
该示例实现一个轻量周期定时器，并支持首个 tick 对齐启动。

## 关键实现点
- `Timer10ms` 提供回调注册、启动、停止等完整生命周期接口。
- 采用独立线程 + `sleep_until` 实现稳定周期调度。
- 支持立即启动（`StartTimeout`）与对齐启动（`StartTimeoutAlignedTo`）。
- 使用原子运行标志与 `join` 保证线程退出安全。
- 将定时器实现打包为静态库，示例程序链接复用。
- 主程序打印 elapsed 时间，便于验证周期与对齐效果。

## 实践价值
- 展示可复用的周期任务组件设计思路，适合迁移到业务定时场景。
