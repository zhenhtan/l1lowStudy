# 实现重点特性

## 目的
该示例用于讲清 C++ 中 `const` 在返回值、参数和成员函数上的不同语义。

## 关键实现点
- 展示返回值前 `const`（对内置值类型意义有限）。
- 展示返回“指向 const 的指针”，防止通过返回值修改内部成员。
- 参数采用 `const 引用`，兼顾“不可改”约束与避免拷贝。
- 成员函数尾部 `const` 表示不修改对象状态，支持 const 对象调用。
- 对比非常量成员函数与常量对象调用限制。
- 包含典型错误示例（注释/演示）帮助理解编译器约束边界。

## 实践价值
- 强化接口 const-correctness，提升可读性与编译期安全性。
