# 实现重点特性

## 目的
该工具将带时间轴的简谱数据渲染为带音轨同步的动态 MP4 视频。

## 关键实现点
- 从 JSON 读取音频、画布、颜色、字体与音符时序配置。
- 通过 `imageio_ffmpeg` 自动定位内置 ffmpeg，降低系统依赖。
- 按行分组音符并自动布局横向坐标，减少人工排版成本。
- 每帧渲染中突出当前音符（高亮+轻微脉冲缩放）。
- 根据当前时刻绘制游标，增强节奏跟随感。
- 使用 MoviePy 组合视频帧与音频轨，完成导出。
- 字体加载带 fallback 机制，提高跨环境可运行性。

## 实践价值
- 完整覆盖“数据建模 -> 布局 -> 动画 -> 编码输出”的媒体生成流程。
