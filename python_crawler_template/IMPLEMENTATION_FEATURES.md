# 实现重点特性

## 目的
该模板提供一个强调合法合规与可运维性的 Python 爬虫最小工程。

## 关键实现点
- 通过 `.env` 管理 URL、选择器、周期、超时、存储路径等配置。
- 可选 robots 校验（默认开启），抓取前先做策略判断。
- `requests` 负责网络访问，包含 UA 与超时控制。
- `BeautifulSoup` 按 CSS 选择器抽取目标内容。
- SQLite 自动建表并持久化抓取记录。
- `schedule` 驱动周期任务，支持长期运行。
- 统一日志输出，方便问题排查与运行监控。

## 实践价值
- 适合作为生产爬虫的入门骨架，便于逐步扩展限流、重试、告警等能力。
