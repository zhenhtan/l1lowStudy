# 实现重点特性

## 目的
该目录将 Python 面试高频点拆分为可独立运行的小示例，便于系统化复习。

## 关键实现点
- 按主题分文件：基础语法、OOP、函数式、迭代器/生成器、asyncio、typing 等。
- 覆盖算法入门题型与常见思路表达。
- 对比线程与进程模型，强调 GIL 与并行适用场景。
- 演示浅拷贝/深拷贝、可变默认参数等常见陷阱。
- 补充实战风格考点：`is None`、排序差异、缓存装饰器、dataclass、Protocol 等。
- 提供 `run_all.py` 一键执行入口，便于整体校验。
- 含链表反转子模块，贴近手写代码题场景。

## 实践价值
- 结构化覆盖面试知识点，支持“按主题刷题 + 统一回归”学习方式。
