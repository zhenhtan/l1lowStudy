# 实现重点特性

## 目的
该目录展示从“入门策略示例”到“增强版回测引擎”的量化实现演进。

## 关键实现点
- 双层示例：`quant_demo.py`（基础均线策略）与 `quant_demo_pro.py`（增强版风控执行）。
- 数据层支持 `yfinance` 在线拉取与离线合成数据回退。
- 策略信号基于短长均线金叉/死叉。
- 增强版考虑手续费、滑点、仓位上限、止损、回撤约束等现实因素。
- 统一 OHLCV 数据结构，保证回测流程可复用。
- 参数化配置将策略逻辑与交易约束分离。

## 实践价值
- 便于理解“教学策略代码”如何向“更接近真实交易约束”的工程实现迁移。
